What is CSS?

CSS stands for Cascading Style Sheets. It is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS defines how elements should be displayed on screen, on paper, or in other media. By using CSS, developers can apply styles to multiple pages of a website, ensuring a consistent look and feel.

Basic Syntax

CSS is composed of rules that are applied to HTML elements. Each rule consists of a selector and a declaration block.

  • Selector: The HTML element you want to style.
  • Declaration Block: Contains one or more declarations separated by semicolons. Each declaration includes a property and a value.
